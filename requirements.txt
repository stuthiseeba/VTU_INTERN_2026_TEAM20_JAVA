DATABASE SETUP
----------------------------------------------------------------

Step 1 — Start MySQL service

Step 2 — Open MySQL Workbench or Command Prompt and run:
  CREATE DATABASE recruitment_system;

Step 3 — Tables are created automatically by Hibernate on
         first run (spring.jpa.hibernate.ddl-auto=update)

Tables created automatically:
  - users         (Team 1 — Auth)
  - drive         (Team 2 — Recruitment, with eligibility fields)
  - student       (Team 2 — with graduationYear and skills)
  - application   (Team 2 — Application workflow)

Step 4 — Optional: load sample data

QUICK START COMMANDS
----------------------------------------------------------------

Backend (Eclipse):
  1. Import project as Maven project
  2. Update Maven (force update)
  3. Right-click PlacementSystemApplication.java
     → Run As → Java Application
  4. Verify: Tomcat started on port 8080

Backend (Command Line):
  cd C:\Users\eknat\VTU_INTERN_2026_TEAM20_JAVA
  mvn clean install
  mvn spring-boot:run

Frontend:
  cd PAT\pat-ui
  npm install
  npm run dev
  Open: http://localhost:5173

Export database snapshot:
  cd "C:\Program Files\MySQL\MySQL Server 8.0\bin"
  mysqldump -u root -p recruitment_system > snapshot.sql

Import database snapshot:
  mysql -u root -p recruitment_system < snapshot.sql
